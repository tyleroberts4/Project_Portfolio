# Project Portfolio

A collection of data science, analytics, and database projects.

## Projects

### [SQL Auto Loan Database](SQL-Auto-Loan-Database-main/)
MySQL database for managing auto loan portfolios with customer, vehicle, accident, and location data. Includes schema, triggers, views, stored procedures, and analytical queries for risk analysis and portfolio reporting.

- **Tech:** MySQL, SQL
- **Highlights:** Relational schema, risk-flag logic, state-level exposure, driver risk ranking, accident hotspots

---

### [Amplytico – Climate-Aware Data Center Optimization](Amplytico-Group-3-main/)
Analysis assets for **Climate-Aware Optimization of Hyperscale Data Center Cooling Systems**. Combines weather data, drought risk, utility pricing, and PUE/WUE simulation for data center cooling optimization.

- **Tech:** Python, Tableau, GIS
- **Highlights:** Open-Meteo weather API, drought risk (USDM), water stress modeling, climate zone analysis, Tableau dashboards

---

### [Frost Risk Forecasting (F3 Innovate)](frost-project-main/)
Probabilistic frost-risk forecasting pipeline for California specialty crop regions using CIMIS weather station data. Predicts frost occurrence and temperature at multiple lead times.

- **Tech:** Python, scikit-learn, XGBoost
- **Highlights:** Feature engineering for early frost detection, LOSO cross-validation, ROC-AUC and Brier score evaluation

---
