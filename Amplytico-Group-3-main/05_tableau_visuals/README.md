# 05 Tableau Visuals

This folder contains the core Tableau workbooks used for the project’s figures and dashboard views.

## Files
- `Counties PUE WUE PRICES.twb`  
  PUE/WUE + pricing + effective cost visuals

- `CountiesWeather.twb`  
  Weather + PUE/WUE exploration

- `Drought Index.twb`  
  Drought index (USDM) visuals and severe drought mapping

- `Electric Price.png`, `Water Price.png`  
  Small supporting images used for labeling/legend content

## Tableau data-link note
The included `*.twb` workbooks were exported from the original workspace and may contain embedded/absolute paths to larger extracts that are not included here.
If Tableau prompts you to reconnect data sources, re-link them to the compact CSVs in `04_analysis_data/` (or to the full datasets from the separate app repository).
